Cloud Native與DevOps系列直播課程

     現今網路世界中，「雲端服務」幾乎是無所不在，實踐這一切的關鍵就是 「虛擬化技術」。傳統雲端應用服務是由實體伺服器（Host）上的「虛擬機 Virtual Machine（VM）」所提供。 應用服務則運行於 VM 之上。

     近年來多雲環境逐漸成熟，強調能讓應用服務快速於不同雲平台遷移的 「軟體貨櫃技術（Container），又稱容器技術」由於運作不會模擬硬體資源， 故相對於 VM，具備更快啟動速度，以及更高可擴充性，逐漸成為企業架構雲端服務的主流技術。

     繼上星期 「虛擬化（Virtual Machine）與容器（Container）架構介紹 」 ，今天我們再次邀請「資策會數位轉型研究所」周育緯工程師來介紹業界 Container Use Case，幫助您認識當前主流的軟體貨櫃解決方案，以及軟體貨櫃叢集管理工具！

課程大綱：

  1. 回顧：Container VS Virtual Machine
  2. Container 使用情境
  3. 業界使用container use case
  4. Container solution 介紹
  5. Container Cluster管理
