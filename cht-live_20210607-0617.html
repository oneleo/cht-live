六月份【微軟 Azure 基礎認證輔導班】課程報名通知

各位同仁好：
　　敬請選修網路管理系列大師 - 楊森盛老師於【06/07 ~ 06/08】（第一梯）、【06/17 ~ 06/18】（第二梯）開辦之【微軟 Azure 基礎認證輔導班】課程。

學習目標：

  1. 了解雲端運算及其關鍵優勢
  2. 了解公有雲、私有雲和混合雲的不同
  3. 分辨 IaaS、PaaS 與 SaaS 服務的差異
  4. 辨別核心的 Azure 架構組件（例如：區域，地理位置以及資源群組）

課程內容：

  1. 雲端運算的概念
  2. 核心的 Azure 服務
  3. 安全性、隱私權、合規性與信任
  4. Azure 定價與支援
  5. Microsoft Azure 基礎知識認證考試複習

授課講師、及您非聽不可的理由：

  1. 微軟認證課程名師楊森盛老師，教學經驗非常豐富。
  2. 本課程市價 12,000 元，錯過可惜，歡迎有興趣同仁踴躍報名參訓。
